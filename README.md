## Cybersecurity Lessons

This project is a simple ASP.NET application focused on the development of cybersecurity solutions. The lessons and exercises included:

Middleware for Access Control – Blocks login attempts from unregistered or unauthenticated users.
Custom CAPTCHA – Random numbers generated and reversed on the server to prevent automated logins.
Decoy Files with Canary Tokens – Files used as traps to detect unauthorized access (currently removed).
Logging System – Tracks user activity and system events for monitoring and auditing.
Automatic Logout Middleware – Logs out users after a period of inactivity to enhance security.
Vigenère Cipher Algorithm – Implemented for license activation of the program.
Validators – Input validation to ensure data integrity and prevent security vulnerabilities.
Identity Management – User authentication, role management, and administration.
Security Settings – Enforces password policies such as minimum length, inclusion of numbers, and uppercase letters.
CRUD Operations for Users and Roles – Admin can create, read, update, and delete users and manage roles.
